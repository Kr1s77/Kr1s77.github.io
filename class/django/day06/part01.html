Serializer序列化器的定义与调用

目标:

  • 了解如何定义 Serializer 序列化器
  • 能够调用定义的 Serializer 序列化器

1. 定义方法

在 DRF 框架中如果需要使用序列化器, 我们需要定义一个序列化器类:

在序列化器类中我们需要指定需要的字段

即:

DRF 中的 Serializer 使用类来定义

须继承自 rest_framework.serializers.Serializer

这里大家需要知道: serializers.ModelSerializer 的父类是 serializers.Serializer

所以我们定义的类, 可以继承自 ModelSerializer 也可以是 Serializer

如果我们继承自 ModelSerializer, 那么它会帮助我们做很多辅助操作.

使用方法: 

from rest_framework import serializers

class UserSerializer(serializers.Serializer):
    '''序列化器类'''
    字段名 = serializers.字段类型(选项参数)

例如: 

我们这里有一个用户类:

class User(object):
    '''用户类'''
    def __init__(self, username, age):
        self.username = username
        self.age = age

需求:

我们希望把这个用户类,转成一个字典: 

{
    "username":"zs",
    "age":18
}

那么, 我们就可以定义一个序列化器类: 

定义序列化器类: 
from rest_framework import serializers

class UserSerializer(serializers.Serializer):
    '''序列化器类'''
    username = serializers.CharField()
    age = serializers.IntegerField()

转化: 

# 创建一个用户类:
user = User('zs', 18)

# 创建一个序列化器类, 并且把用户类传入进去:
serializer = UserSerializer(user)

# 可以通过data查看结果
serializer.data

验证:

我们单独定义一个 test.py 文件,测试一下上面的代码是否可行:

代码:

再例如

我们再新建一个序列化文件演示以下功能:

  1. 创建 serializers.py 文件
# 导入:
from rest_framework import serializers

class BookInfoSerializer(serializers.Serializer):
    """图书数据序列化器"""
    btitle = serializers.CharField(label='名称', max_length=20)
    bpub_date = serializers.DateField(label='发布日期')
    bread = serializers.IntegerField(label='阅读量')
    bcomment = serializers.IntegerField(label='评论量')
    image = serializers.ImageField(label='图片')

参数解读:

label 指的是在上一小节的可视化界面中,我们看到的提示信息. 可以由这里的label指定. 和 verbose_name很像.

  1. 定义模型文件: models.py
class BookInfo(models.Model):
    btitle = models.CharField(max_length=20, verbose_name='名称')
    bpub_date = models.DateField(verbose_name='发布日期', null=True)
    bread = models.IntegerField(default=0, verbose_name='阅读量')
    bcomment = models.IntegerField(default=0, verbose_name='评论量')
    image = models.ImageField(upload_to='booktest', verbose_name='图片', null=True)

3. 创建Serializer对象

定义好Serializer类后，就可以创建Serializer对象了。

Serializer的构造方法为：

Serializer(instance=None, data=empty, **kwarg)

说明：

1）用于序列化时，将模型类对象传入instance参数

2）用于反序列化时，将要被反序列化的数据传入data参数

3）除了instance和data参数外，在构造Serializer对象时，还可通过context参数额外添加数据，如

serializer = AccountSerializer(account, context={'request': request})

通过context参数附加的数据，可以通过Serializer对象的context属性获取。
