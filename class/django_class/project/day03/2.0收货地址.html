Introduction

收货地址

学习目标:

  • 地址页面我们需要做的业务逻辑

作用:

  • 能够对后面的业务逻辑有一个全局的认识

收货地址页面:

在这个页面中，我们要实现用户地址的管理，主要的业务逻辑有：

  • 省市区地址的数据库建立与查询
  • 用户地址的增删改查处理
  • 设置默认地址
  • 设置地址标题

其中, 我们主要讲解省市区地址的三级联动和缓存，其余留给大家自己来实现
