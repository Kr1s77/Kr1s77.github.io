获取订单结算页面数据---接口

订单结算页面所需的数据从购物车中勾选而来。

说明:

  • 订单页面中的地址部分, 是从地址接口获取的, 这部分我们不用管
  • 我们这里需要做的是订单下面的接口实现.

1. 后端接口设计

请求方式 ： GET /orders/settlement/

请求参数： 无

返回数据： JSON

参数 类型 是否必须 说明
freight decimal 是 运费
skus sku[] 是 结算的商品列表
id int 是 商品id
name str 是 商品名称
default_image_url str 是 商品默认图片
price decimal 是 商品单价
count int 是 商品数量

2. 后端实现

在 orders / views.py 中编写视图


class OrderSettlementView(APIView):
    """订单结算接口"""

    # 当前接口需要认证过得用户才能进行访问: 
    permission_classes = [IsAuthenticated]

    # 我们定义一个接口, 让用户能够访问到, 并获取数据
    # 这里不能简写的原因: 是因为我们是从redis中获取,而不是从mysql数据库中获取结果: 
    # 所以 get 函数不能省略
    def get(self, request):
        """获取订单结算信息"""

        # 拿到认证的用户
        user = request.user

        # 1. 从redis中获取用户勾选的商品, 这些商品就是结算的商品:
        # 1.1 链接 cart redis:
        redis_conn = get_redis_connection('cart')

        # 1.2 通过key, 获取redis中所有勾选的商品:
        sku_ids = redis_conn.smembers('cart_selected_%s' % user.id)

        # 1.3 通过key, 获取redis中商品的sku_id和数量(count)
        redis_cart = redis_conn.hgetall('cart_%s' % user.id)

        cart = {}
        # 2 遍历, 获取商品id 和 数量(count)
        for sku_id, count in redis_cart.items():
            # 整理数据: 
            # 2.1 将选中的商品的sku_id 和 数量 保存到一个字典中
            cart[int(sku_id)] = int(count)

        # 3. 查询出所有的勾选的商品, 即要计算的商品信息.
        # 这里不需要转换, 系统帮我们进行了转换: 
        skus = SKU.objects.filter(id__in=sku_ids)

        # 3.1 遍历这些商品, 获取每一个
        for sku in skus:
            # 给每个sku商品, 增加上 count 属性: 
            sku.count = cart[sku.id]

        # 我们自定义一个运费值, 这个数是随意定的, 在实际开发中
        # 这个数值,可以定义一个单独的表:
        # Decimal: 代表定义的是十进制的浮点数
        freight = Decimal(10)

        # 4 序列化数据, 并进行返回: 
        # 4.1 我们可以定义一个序列化器, 对数据进行序列化操作, 然后返回结果:
        serializer = OrderSettlementSerializer({'freight': freight, 'skus': skus})

        # 将最终结果返回: 
        return Response(serializer.data)

在 orders / serialziers.py 中创建序列化器

# 我们需要定义第一个序列化器, 因为这个序列化器还能够对 运费值 进行序列化操作
class OrderSettlementSerializer(serializers.Serializer):
    """
    订单结算数据序列化器
    """
    # 定义序列化的两个字段: 
    # 第一个字段是运费数据, 指明当前运费数据是十进制类型的小数.
    # 最大位数为10位, 小数点后有2位
    freight = serializers.DecimalField(label='运费', max_digits=10, decimal_places=2)
    # 第二个字段是商品, 这个字段还是使用上面的序列化器进行序列化:
    skus = CartSKUSerializer(many=True)

创建的第二个序列化器: 

# 序列化器我们定义了两个: 
# 第二个序列化器是对购物车中的商品进行的序列化操作:
class CartSKUSerializer(serializers.ModelSerializer):
    """
    购物车商品数据序列化器
    """
    # 数量这个字段,在模型类中没有, 所以我们单独定义出来
    count = serializers.IntegerField(label='数量')

    class Meta:
        model = SKU
        fields = ('id', 'name', 'default_image_url', 'price', 'count')
路由:
# 总路由: 
url(r'^', include('orders.urls'))

# 子路由:

新建 place_order.js
var vm = new Vue({
    el: '#app',
    data: {
        host,
        username: sessionStorage.username || localStorage.username,
        user_id,
        token,
        skus: [],
        freight: 0, // 运费
        total_count: 0,
        total_amount: 0,
        payment_amount: 0,
        order_submitting: false, // 正在提交订单标志
        pay_method: 1, // 支付方式,
        nowsite:0, // 默认地址
        addresses: []
    },
    mounted: function(){
        // 获取地址信息
        axios.get(this.host + '/addresses/', {
                headers: {
                    'Authorization': 'JWT ' + this.token
                },
                responseType: 'json'
            })
            .then(response => {
                this.addresses = response.data.addresses;
                this.nowsite = response.data.default_address_id;
            })
            .catch(error => {
                status = error.response.status;
                if (status == 401 || status == 403) {
                    location.href = 'login.html?next=/user_center_site.html';
                } else {
                    alert(error.response.data.detail);
                }
            })
        // 获取结算商品信息
        axios.get(this.host+'/orders/settlement/', {
                headers: {
                    'Authorization': 'JWT ' + this.token
                },
                responseType: 'json'
            })
            .then(response => {
                this.skus = response.data.skus;
                this.freight = response.data.freight;
                this.total_count = 0;
                this.total_amount = 0;
                for(var i=0; i<this.skus.length; i++){
                    var amount = parseFloat(this.skus[i].price)*this.skus[i].count;
                    this.skus[i].amount = amount.toFixed(2);
                    this.total_count += this.skus[i].count;
                    this.total_amount += amount;
                }
                this.payment_amount = parseFloat(this.freight) + this.total_amount;
                this.payment_amount = this.payment_amount.toFixed(2);
                this.total_amount = this.total_amount.toFixed(2);
            })
            .catch(error => {
                if (error.response.status == 401){
                    location.href = '/login.html?next=/cart.html';
                } else{
                    console.log(error.response.data);
                }
            })
    },
    methods: {
        // 退出
        logout: function(){
            sessionStorage.clear();
            localStorage.clear();
            location.href = '/login.html';
        },
        // 提交订单
        on_order_submit: function(){

        }
    }
});
